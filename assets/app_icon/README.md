# Uygulama ikonu

- `app_icon.png` — 1024×1024 kaynak (UNIQ sari U, siyah zemin)
- Degistirmek: PNG'yi degistirin, sonra proje kokunde:

```powershell
dart run flutter_launcher_icons
```

- Yeniden uretmek: `scripts\generate-app-icon.ps1`

Yeni ikon App Store'da ancak **yeni build yuklendikten** sonra gorunur.
